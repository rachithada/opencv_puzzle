# OpenCV Live Camera Puzzle Game

This is an interactive OpenCV-based puzzle game where the user can use the live camera feed to select an area by drawing a rectangle. The selected region is then converted into a 3x3 puzzle grid.

The project uses OpenCV for camera handling and image processing, and MediaPipe for hand landmark detection.

## Features

- Live webcam-based interaction
- Draw rectangle on live camera feed
- Capture selected region from the frame
- Convert selected region into a 3x3 puzzle
- Hand gesture based interaction using MediaPipe
- Drag and drop puzzle pieces
- Real-time puzzle gameplay

## Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
